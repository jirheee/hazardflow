## HazardFlow HDL User Guide

```bash
# Install Rust and mdBook
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
cargo install mdbook

# Open browser
# If you're working on Coder, see <https://coder.com/docs/networking/port-forwarding#dashboard>.
mdbook serve --open

# Build
mdbook build
```
